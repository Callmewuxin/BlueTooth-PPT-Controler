BlueTooth-PPT-Controler
=======================

The android app is use to control the power point through bluetooth .
The BlueTooth is the app source file , the server directory is the PC server source file .

++++++++++++++++++++++++++++++++++++
这是一个利用蓝牙来控制PPT的小android应用。
BlueTooth 是手机端的应用的源代码
Server文件夹是电脑服务端的源代码。
